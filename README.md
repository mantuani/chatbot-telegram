# chatbot-telegram
a simple chatbot, like a digital secretary.

-english version below

Esse projeto é um bot para Telegram que criei para testar automação e integração com a API do Telegram. A ideia é que o bot responda comandos enviados no chat e execute algumas ações automaticamente.

Foi desenvolvido como forma de aprendizado e prática com Python e bots.

-- Como funciona:

O bot fica rodando em segundo plano e escuta as mensagens enviadas no chat.
Quando um comando é enviado, ele identifica esse comando e responde de acordo com o que foi programado.

Novas funções podem ser adicionadas facilmente no código.

-- Como usar:

Para usar o bot, é necessário criar um bot no Telegram usando o BotFather.
Depois de criar, o Telegram fornece um token, que deve ser colocado no código do projeto.

Com o token configurado, basta executar o arquivo principal em Python para iniciar o bot.

Tecnologias usadas:

>Python,
>Telegram Bot API

________________________________________________________________________________________________________________
This project is a Telegram bot created to test automation and integration with the Telegram API. The idea is for the bot to respond to commands sent in the chat and perform certain actions automatically.

It was developed as a way to learn and practice working with Python and bots.

--How it works:

The bot runs in the background and listens for messages sent in the chat.
When a command is received, it identifies the command and responds according to the logic implemented in the code.

New features can be added easily by modifying the source code.

--How to use:

To use the bot, you first need to create a bot on Telegram using BotFather.
After creating it, Telegram provides a token that must be inserted into the project code.

Once the token is configured, simply run the main Python file to start the bot.

Technologies used:

>Python,
>Telegram Bot API
